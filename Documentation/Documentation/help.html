Set up Email Adrress for Contact & Subscribe Form

To set up the contact form to send emails with the persons which are contacting you, open "contact.php" file and change the email from line 52 with your own email. 

Line 52: $address = "example@example.net";

To set up the subscribe form to send emails with the persons which are contacting you, open "subscribe.php" file and change the email from line 26 with your own email. 

Line 26: $address = "example@example.net";

 

Set up Google map

Go to https://maps.google.com/


    var options = {
	mapTypeControlOptions: {
	mapTypeIds: ['Styled']
	},
	center: new google.maps.LatLng(-7.245217594087794, 112.74455556869509),
	zoom: 16,
	disableDefaultUI: true,	
	scrollwheel: false,
	mapTypeId: 'Styled'
};
